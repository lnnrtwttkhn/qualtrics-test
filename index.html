Code Highlighting

Add external CSS

  1. In the Survey tab, click on Look and feel.
  2. Click on Style.
  3. Paste the following link into the External CSS field.
  4. Don’t forget to click Apply (bottom right) to apply your changes.
https://cdnjs.cloudflare.com/ajax/libs/prism/1.23.0/themes/prism.min.css

Add JavaScript to header

  1. In the Survey tab, click on Look and feel.
  2. Click on General.
  3. In the field Header, select My Library: Your Name (Your Name should display your name).

Create new message

If you want to add JavaScript libraries for the first time:

  1. Select New Message …. A window Create a New Message should open.
  2. For the Category select Look and Feel Messages
  3. For the Description pick a description. For example, “Import JavaScript Libraries”.
  4. In the editor window, select Source
  5. Paste the following HTML code:
  6. Don’t forget to click Apply (bottom right) to apply your changes.
<script src="https://cdnjs.cloudflare.com/ajax/libs/prism/1.23.0/prism.min.js"></script>
<script src="https://cdnjs.cloudflare.com/ajax/libs/prism/1.23.0/components/prism-bash.min.js"></script>

Note, that you might need to remove the newline for this to work.

Add existing message

  1. In the field Header, select My Library: Your Name (Your Name should display your name).
  2. Select an existing message. For example, “Import JavaScript Libraries”.
  3. Don’t forget to click Apply (bottom right) to apply your changes.

Add code block

To add a code block to the question text or a choice option, use the following code snippet.

<pre class="language-bash"><code>git status</code></pre>

Note, that you might need to enter the HTML View.

For multi-line code blocks, add newlines like this:

<pre class="language-bash"><code>git status
git add .
git commit -m "add code"</code></pre>

Import survey

It’s possible to import survey questions containing code blocks to Qualtrics using a simple .txt-file. For details on survey import from .txt-files, see this section of the Qualtrics Support.

survey-code-block.txt
[[AdvancedFormat]]

[[Block:Default Question Block]]

[[Question:MC]]
[[ID:Q1]]
Let's see if we can import survey questions with code.
<pre class="language-bash"><code>wget code-block-survey.txt</code></pre>

[[AdvancedChoices]]
[[Choice]]
<pre class="language-bash"><code>git status</code></pre>
[[Choice]]
<pre class="language-bash"><code>git status
git add .
git commit -m "add code"</code></pre>
[[Choice]]
<pre class="language-bash"><code>git pull
git push
git merge main</code></pre>
